# Find Tractable Multitree Recolorabilities

## Build

You can build FTMR library using CMake.

```
cmake -S . -B build
cmake --build build
```

## Test

Go to the `build/` directory and execute `ctest`.

```
cd build/ && ctest
```

You can also execute the test directly.

```
./build/test/unit_test
```
